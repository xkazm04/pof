# Parameter sensitivity sweep (tornado chart)

## Description
Add a sweep mode that varies one chosen parameter (e.g. enemy-kill-gold baseAmount or drops/hr) across a range, re-runs the deterministic seeded sim at each step, and plots the effect on key outputs (net flow, Gini, rarity inflation) as a ranked tornado/line chart showing which knobs move the economy most.

## Reasoning
Designers rarely know which of dozens of parameters dominates an imbalance. Sensitivity analysis (a staple of financial tornado diagrams and Machinations sweeps) points them straight at the highest-leverage knob, sharply shortening tuning cycles, and the existing seeded RNG makes the runs reproducible and comparable.
