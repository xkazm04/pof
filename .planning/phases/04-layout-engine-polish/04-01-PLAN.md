**A. Density crossfade (DENS-13):**
In each panel, wrap the density switch output (the `{density === 'micro' && ...}` block) with `AnimatePresence mode="wait"` containing a `motion.div` keyed by density value. Use `DZIN_TIMING.DENSITY / 2` for fade in/out duration (0.1s each = 0.2s total crossfade). Import `AnimatePresence`, `motion` from `framer-motion` and `DZIN_TIMING` from `@/lib/dzin/animation-constants`.

Pattern for each panel:
```tsx
<AnimatePresence mode="wait">
  <motion.div
    key={density}
    initial={{ opacity: 0 }}
    animate={{ opacity: 1 }}
    exit={{ opacity: 0 }}
    transition={{ duration: DZIN_TIMING.DENSITY / 2 }}
  >
    {density === 'micro' && <MicroView ... />}
    {density === 'compact' && <CompactView ... />}
    {density === 'full' && <FullView ... />}
  </motion.div>
</AnimatePresence>
```
